# TODO APP

## Installation

1. Clone the repository

   ```
   git clone  https://github.com/iam-abin/todo-list-reactjs.git
   ```

2. Navigate to the project directory

   ```
   cd todo-list-reactjs
   ```

3. Install the dependencies

   ```
   npm install 
   ```

4. Start the application

   ```
   npm start
   ```
